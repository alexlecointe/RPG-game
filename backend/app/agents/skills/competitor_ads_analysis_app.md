# Skill: Analyse pubs concurrents — App mobile

Applique les instructions generiques de competitor_ads_analysis, plus ces specificites app mobile :

## Analyse supplementaire

### Analyse store ads
- Apple Search Ads competitors (keywords biddes, brand vs generic)
- Keyword bidding (termes cibles, estimations CPA)
- Screenshot A/B tests (variations detectees sur les fiches store)

### Analyse UA
- Canaux user acquisition : TikTok, Meta, Google UAC
- Influenceurs (micro vs macro, formats sponsorises)
- Repartition estimee du budget par canal

### Creative app
- Video gameplay/demo (duree, hook, CTA)
- Screenshot animee (motion design, transitions)
- UGC (temoignages utilisateurs, screen recordings)
- Testimonial (format face camera, overlay resultats)

### CPI benchmark
- Cout par install moyen du secteur (par plateforme)
- Evolution du CPI sur les 12 derniers mois
- Saisonnalite (pics et creux de cout d'acquisition)
