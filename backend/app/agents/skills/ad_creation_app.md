# Skill: Creation publicitaire — App mobile

Applique les instructions generiques de ad_creation, plus ces specificites app mobile :

## Analyse supplementaire

### Formats app
- Video app preview (15-30s, flow utilisateur cle)
- Screenshot animee (transitions entre ecrans phares)
- Screen recording (utilisation reelle avec annotations)
- App Install ads (format natif iOS/Android avec bouton install)

### Copy app
- Nombre de telechargements comme preuve sociale ("500K+ utilisateurs")
- Note store mise en avant (4.8★ sur l'App Store)
- "Gratuit" ou "Essai gratuit" en accroche principale
- Fonctionnalite star en une phrase (le benefice #1)

### Visuels app
- Mockup device (iPhone/Android avec ecran app visible)
- UI animation (micro-interaction ou feature cle en motion)
- Before/after experience (sans l'app vs avec l'app)
- User testimony video (temoignage face camera + overlay UI)

### App-specific
- Apple Search Ads (keywords competiteurs, brand terms)
- Google UAC (Universal App Campaigns, assets multiples)
- TikTok App Install (format vertical natif, hook 3s)
- Deep link retargeting (ramener les utilisateurs inactifs sur un ecran precis)
