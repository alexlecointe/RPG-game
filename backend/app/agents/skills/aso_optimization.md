# Skill: Optimisation ASO (App Store Optimization)

## Objectif
Optimiser la fiche App Store / Google Play pour maximiser les telechargements organiques.

## Structure attendue

### 1. Recherche de mots-cles
- 30 mots-cles principaux classes par volume et difficulte
- Mots-cles utilises par les concurrents directs
- Mots-cles longue traine a faible concurrence
- Tableau : | Mot-cle | Volume estime | Difficulte | Pertinence |

### 2. Titre de l'app
- Titre optimise (max 30 caracteres iOS, 50 Android)
- 3 variantes a tester
- Mot-cle principal integre naturellement

### 3. Sous-titre / Short description
- Sous-titre iOS optimise (max 30 caracteres)
- Short description Android (max 80 caracteres)
- Proposition de valeur claire + mot-cle

### 4. Description longue
- Description complete optimisee (4000 caracteres max)
- Structuree avec bullet points et emojis strategiques
- Mots-cles integres naturellement (pas de keyword stuffing)
- Sections : intro hook, fonctionnalites cles, social proof, CTA

### 5. Keyword field (iOS uniquement)
- 100 caracteres de mots-cles (separes par virgule, sans espaces)
- Pas de repetition avec le titre
- Singulier uniquement (Apple indexe le pluriel)

### 6. Strategie screenshots
- 6-10 screenshots recommandes avec texte overlay pour chacun
- Ordre optimise (premier screenshot = plus impactant)
- Texte court par screenshot (benefice, pas fonctionnalite)

### 7. Plan d'iteration
- Metriques a suivre (impressions, conversion rate, keyword rankings)
- Frequence de mise a jour recommandee
- Tests A/B a prioriser
