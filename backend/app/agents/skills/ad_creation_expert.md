# Skill Expert: Creation Publicitaire Multi-Plateforme

## Objectif
Produire un kit publicitaire complet et pret a deployer sur Meta Ads, TikTok Ads et Google Ads, avec des creatives differenciees par angle strategique. Chaque creative doit respecter les specs techniques de sa plateforme, proposer un hook unique, et inclure un brief visuel actionnable pour la production. Le kit doit aussi inclure un plan de test A/B structure.

## Framework
**AIDA + PAS (Problem-Agitate-Solve)** + bonnes pratiques plateforme-specifiques.

Deux frameworks de copywriting complementaires pour maximiser la diversite des angles :

- **AIDA (Attention, Interest, Desire, Action)** : structure classique orientee benefice. Le hook capte l'attention, le corps construit l'interet et le desir, le CTA declenche l'action. Ideal pour les angles "benefice" et "social proof".
- **PAS (Problem, Agitate, Solve)** : structure orientee douleur. On nomme le probleme, on l'amplifie emotionnellement (agitate), puis on presente la solution comme soulagement. Ideal pour les angles "douleur" qui convertissent fort en retargeting.

Chaque kit doit contenir au minimum 3 angles differents pour permettre un A/B test significatif. Un seul angle = pas de learning.

## Structure attendue

### 1. Brief creatif
- **Objectif de la campagne** : awareness (notoriete), consideration (interet), conversion (achat/inscription)
- **Persona cible** : 1 persona precis avec demographics, psychographics, frustrations quotidiennes, et ou il/elle passe du temps online
- **Message cle** : la proposition de valeur en une phrase
- **Tone of voice** : formel/casual/urgent/inspirant — avec un exemple de phrase dans le ton
- **Contrainte budgetaire** : si connue, adapter le nombre de variantes et le plan de test

### 2. Framework creatif : 3 angles obligatoires

Chaque kit doit explorer 3 angles distincts. Chaque angle doit etre utilise au moins une fois dans le kit :

#### Angle Douleur (PAS)
- **Problem** : nommer la frustration specifique de la cible en 1 phrase
- **Agitate** : amplifier avec des consequences concretes ("Et pendant ce temps, vos concurrents...")
- **Solve** : presenter la solution comme le soulagement naturel

#### Angle Benefice (AIDA)
- **Attention** : hook surprenant ou chiffre impactant
- **Interest** : "Comment [persona] a obtenu [resultat] en [temps]"
- **Desire** : details du resultat, projection dans le succes
- **Action** : CTA urgent mais ethique

#### Angle Social Proof
- **Temoignage** : citation reelle ou realiste d'un client avec resultat chiffre
- **Chiffre d'autorite** : nombre d'utilisateurs, note, croissance
- **Transfert de confiance** : logo, mention presse, certification

### 3. Creatives Meta Ads

Produire 3 creatives minimum, chacune avec un angle different :

Pour chaque creative :
- **Format** : image 1080x1080 (feed) ou video 9:16 (story/reel)
- **Primary text** : 125 caracteres visibles avant "...voir plus", 250 caracteres max total
- **Headline** : 40 caracteres maximum
- **Description** : 30 caracteres maximum
- **CTA button** : choisir parmi Learn More, Shop Now, Sign Up, Get Offer, Book Now, Download, Contact Us
- **Brief visuel** : description detaillee de ce que l'image/video doit montrer (composition, couleurs, texte overlay, ambiance) — pas juste "image du produit"
- **Angle utilise** : douleur, benefice, ou social proof

Specs a respecter :
- Primary text : ne PAS depasser 250 caracteres
- Headline : ne PAS depasser 40 caracteres
- Description : ne PAS depasser 30 caracteres
- Ratio image : 1:1 (feed) ou 9:16 (stories)

### 4. Creatives TikTok Ads

Produire 2 scripts minimum, style UGC (User Generated Content) :

Pour chaque script :
- **Hook (0-3 secondes)** : pattern interrupt — question choc, affirmation contre-intuitive, ou "Arretez de faire [X]". Le hook doit etre UNIQUE par script, jamais le meme
- **Body (3-20 secondes)** : developper le probleme ou le benefice selon l'angle choisi
- **CTA (20-25 secondes)** : appel a l'action clair avec urgence legere
- **Format** : face-to-camera UGC, text overlay, ou screen recording selon le produit
- **Notes de production** : transitions recommandees, musique trending, texte overlay exact, cadrage
- **Angle utilise** : douleur, benefice, ou social proof

Specs a respecter :
- Duree : 15-30 secondes maximum
- Format : vertical 9:16 exclusivement
- Les 3 premieres secondes sont critiques — le hook decide du watch rate
- Style natif TikTok (pas de pub "corporate")

### 5. Creatives Google Ads

Produire 3 Responsive Search Ads minimum :

Pour chaque RSA :
- **15 headlines** : 30 caracteres maximum chacun, varier les angles et inclure des keywords
- **4 descriptions** : 90 caracteres maximum chacune
- **Pin strategy** : quels headlines pinnen en position 1/2/3 et pourquoi
- **Extensions recommandees** :
  - Sitelinks (4 minimum) : titre + description + URL
  - Callouts (4 minimum) : arguments courts (25 chars max)
  - Structured snippets : categorie + valeurs

Specs a respecter :
- Headline : 30 caracteres max STRICT (inclure espaces)
- Description : 90 caracteres max STRICT
- Callout : 25 caracteres max
- Varier les headlines : inclure le nom de marque, des benefices, des chiffres, des CTAs, des keywords
- Ne PAS repeter le meme message dans plusieurs headlines

### 6. Plan de test A/B

#### Variables a tester
- **Creative** : image vs video, lifestyle vs product-shot
- **Hook** : douleur vs benefice vs social proof
- **Audience** : broad vs lookalike vs interest-based
- **CTA** : "Essayer gratuitement" vs "Voir la demo" vs "Commander maintenant"

#### Metriques de decision
- **Meta** : CTR > 1%, CPC < objectif, ROAS > 2x
- **TikTok** : Watch rate > 25% a 3s, CTR > 0.8%, CPA < objectif
- **Google** : CTR > 3%, Quality Score > 7, CPC < objectif secteur

#### Protocole de test
- Budget minimum par variante : €10-20/jour pendant 5-7 jours
- Condition de significativite : minimum 100 clics par variante avant de conclure
- Processus : tester le hook d'abord (3-5 jours), puis l'audience (3-5 jours), puis le CTA (3-5 jours)
- Kill criterion : si CTR < 0.5% apres 48h et 1000+ impressions, arreter la variante

## Criteres de qualite (score 10/10)
- **3+ creatives avec specs completes par plateforme** : chaque creative a tous les champs remplis (copy, CTA, brief visuel, format)
- **Angles distincts par variante** : chaque creative utilise un angle different (douleur, benefice, proof) — pas 3 fois le meme message reformule
- **Copy respecte les limites de caracteres** : headline Meta ≤ 40 chars, description ≤ 30 chars, Google headline ≤ 30 chars, description ≤ 90 chars
- **Brief visuel/image pour chaque creative** : description detaillee de la direction artistique, pas juste "mettre une image du produit"
- **CTA explicites et varies** : chaque creative a un CTA adapte a son angle et a la plateforme

## Exemple de sortie

```markdown
### Meta Ad — Creative 1 (Angle Douleur / PAS)

**Format** : Image 1080x1080 (feed)

**Primary text** (198 chars) :
Vous perdez encore 2h par jour a relancer vos prospects manuellement ?
Pendant ce temps, vos concurrents automatisent. ReachBot envoie les bons
messages au bon moment. Resultats des la premiere semaine.

**Headline** (38 chars) : Automatisez votre prospection B2B

**Description** (28 chars) : Essai gratuit — 14 jours

**CTA button** : Sign Up

**Brief visuel** : Split screen — a gauche, un bureau encombre avec
post-its et tableur Excel (ambiance stressante, couleurs froides).
A droite, un dashboard epure ReachBot avec des notifications
"Nouveau lead qualifie" (ambiance sereine, couleurs chaudes).
Texte overlay en bas : "Avant / Apres ReachBot".
```

## Erreurs a eviter
- **Meme angle repete 3 fois** : 3 variantes "douleur" = pas de test valide. Varier systematiquement les angles
- **Copy qui depasse les limites** : compter les caracteres AVANT de finaliser — les plateformes tronquent sans pitie
- **Pas de hook dans les 3 premieres secondes** : sur TikTok, 65% des viewers partent avant 3 secondes — le hook est non-negociable
- **CTA vague** : "En savoir plus" partout est paresseux — adapter le CTA a l'angle et au stade du funnel
- **Visuels non decrits** : "Image du produit" n'est pas un brief — decrire la composition, les couleurs, le texte overlay, l'ambiance
- **Ignorer les specs par plateforme** : un RSA Google n'est pas un post Meta — respecter les formats et les contraintes de chaque canal
- **Claims non verifiables** : "N°1 du marche", "La meilleure solution" — si ce n'est pas prouvable, ne pas l'ecrire

## Regles
- Chaque creative doit avoir un angle distinct parmi : douleur (PAS), benefice (AIDA), social proof
- Les limites de caracteres par plateforme sont strictes et non negociables
- Chaque creative doit inclure un brief visuel detaille (pas juste "image du produit")
- Le plan de test doit inclure des metriques de decision chiffrees
- Aucun claim non verifiable ou exagere dans les copies
- Le kit doit couvrir les 3 plateformes (Meta, TikTok, Google) sans exception
