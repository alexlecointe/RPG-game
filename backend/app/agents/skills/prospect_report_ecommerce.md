# Skill: Rapport prospection — E-commerce

Applique les instructions generiques de prospect_report, plus ces specificites e-commerce :

## Analyse supplementaire

### Prospects e-commerce
- Influenceurs produit (micro et macro, niche relevante)
- Revendeurs potentiels (boutiques physiques, e-shops complementaires)
- Marketplaces niche (specialisees par categorie produit)
- Boutiques complementaires (produits associes, meme audience)

### Criteres e-commerce
- Volume audience (followers, trafic mensuel, liste email)
- Taux d'engagement (likes, commentaires, partages)
- Niche match (alignement categorie produit et audience)
- Historique collabs (marques similaires, formats testes)

### Canaux e-commerce
- DM Instagram (approche directe, message personnalise)
- Email PR (press kit, proposition de valeur claire)
- Plateformes influenceurs (Upfluence, Aspire, Heepsy)
- Salons pro (rencontres physiques, networking sectoriel)

### Partenariats
- Co-branding (produit ou collection en commun)
- Bundles (offre groupee avec marque complementaire)
- Affiliate (commission sur ventes generees)
- Guest sur newsletter (echange de visibilite audience)
