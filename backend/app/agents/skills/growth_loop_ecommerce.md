# Skill: Growth loop — E-commerce

Applique les instructions generiques de growth_loop, plus ces specificites e-commerce :

## Analyse supplementaire

### Referral e-commerce
- Programme parrainage avec credit ou remise pour les deux parties
- Programme ambassadeur pour les clients les plus fideles
- Experience unboxing virale (packaging Instagram-friendly)
- Codes promo partageables et trackables

### Repeat purchase loop
- Email post-achat (merci, conseils utilisation, cross-sell)
- Recommandations cross-sell basees sur l'historique
- Programme fidelite a points (achat = points = reductions)
- Restock reminders pour les produits consommables

### UGC loop
- Avis photo/video contre remise sur la prochaine commande
- Hashtag de marque encourage sur les reseaux sociaux
- Repost contenu client sur les comptes officiels
- Concours photo/video reguliers

### Marketplace expansion
- Presence sur Amazon, Etsy et autres marketplaces pertinentes
- Multi-canal comme levier de decouverte de la marque
- Strategie de prix coherente entre canaux
- Redirection marketplace vers le site propre pour fideliser
