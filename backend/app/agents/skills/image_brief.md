# Skill: Brief visuel

## Objectif
Produire un brief creatif complet pour la creation d'images marketing ou produit. Le brief doit etre suffisamment precis pour qu'un designer ou un outil IA puisse produire le visuel sans questions supplementaires.

## Structure attendue

### 1. Direction artistique
- Style : photo, illustration, 3D, flat design, pixel art, etc.
- Mood board : 3 a 5 references concretes (URLs, noms de marques, campagnes connues)
- Palette couleur : couleurs principales (hex), couleurs secondaires, couleurs interdites
- Ambiance : mots-cles d'atmosphere (ex : chaleureux, tech, minimaliste, energique)

### 2. Specs techniques
- Dimensions par plateforme :
  - Instagram : 1080x1080 (feed), 1080x1920 (story)
  - Meta Ads : 1200x628
  - Web : 1920x1080
  - Autres si applicable
- Format fichier : PNG, JPG, SVG, WebP
- Resolution : 72dpi (web) ou 300dpi (print)
- Poids max si applicable

### 3. Contenu visuel
- Sujet principal : ce qui doit etre au centre de l'image
- Composition : disposition des elements, zone de focus, espace negatif
- Texte overlay : texte exact si applicable, police suggeree, taille relative
- Elements de marque : logo, baseline, charte couleur a respecter

### 4. Variantes
- 3 variantes minimum avec angles ou contextes differents
- Pour chaque variante : description, usage prevu, ce qui change vs la version principale
- Indiquer la variante recommandee comme principale

### 5. Usage
- Ou chaque image sera utilisee (canal, emplacement)
- Comment elle sera utilisee (hero image, thumbnail, ad creative, social post)
- Contexte d'affichage (fond clair/sombre, a cote de quel contenu)

## Regles
- References concretes obligatoires : pas de "quelque chose de moderne" mais "dans le style de la campagne Apple Shot on iPhone 2024"
- Specs techniques obligatoires pour chaque livrable
- Coherent avec la charte de marque si elle existe
- Aucune description vague : chaque element doit etre actionnable par un createur
