# Skill: Landing page — E-commerce

Applique les instructions generiques de landing_page, plus ces specificites e-commerce :

## Direction artistique par categorie produit

Detecte la categorie du produit depuis le brief et applique le systeme visuel correspondant :

### Wellness / Skincare / Sante (creme, supplement, soin, nutrition)
- Palette : fond creme/ivoire, vert sauge ou vert foret, accent abricot/corail doux
- Typo : serif editorial pour les titres (Playfair Display, DM Serif), sans-serif clean pour le corps
- Style photo : lumiere naturelle douce, surface marbre ou pierre, fond neutre, ombres subtiles
- References : Aesop, Byredo, Dr. Hauschka, Lush, Ceremonie
- Ton : premium, naturel, rassurant, science + nature

### Sport / Fitness / Active (equipement, vetement, performance)
- Palette : noir profond ou marine, accent electrique (vert neon, orange vif, rouge)
- Typo : sans-serif bold condensed (Bebas Neue, Anton, Barlow Condensed)
- Style photo : action shot, athlete en mouvement, eclairage dramatique
- References : Nike, Under Armour, Gymshark, Satisfy Running
- Ton : energique, ambitieux, high-performance

### Food / Boisson / Snack (produit alimentaire, boisson, condiment)
- Palette : tons chauds (orange, jaune miel, rouge brique) ou frais (vert vif, citron)
- Typo : humaniste, friendly, lisible (Nunito, Raleway, Cabin)
- Style photo : overhead shot, ingredients etales, lifestyle repas
- References : Oatly, Tony's Chocolonely, Innocent, Graze
- Ton : fun, transparent, engagement, origine tracee

### Mode / Accessoires / Lifestyle (vetement, bijou, maroquinerie)
- Palette : blanc casse + noir + 1 accent luxe (or, bordeaux, bleu nuit)
- Typo : serif fin (Cormorant, Garamond) ou sans minimaliste (Helvetica, Inter)
- Style photo : editorial magazine, mannequin ou flat lay, fond uni
- References : Jacquemus, A.P.C., Sezane, Veja
- Ton : curated, desirable, storytelling

### Tech / Gadget / Home (appareil, outil, objet connecte)
- Palette : blanc, gris anthracite, 1 accent couleur vive
- Typo : sans-serif moderne (Inter, DM Sans, Manrope)
- Style photo : produit sur fond blanc ou gradient, close-up detail, lifestyle
- References : Apple, Braun, Sonos, Withings
- Ton : clair, benefit-driven, technologie accessible

## Analyse supplementaire

### Conversion e-commerce
- Hero avec produit star en grand visuel haute qualite (image generee ou asset existant)
- Prix visible et clair au-dessus du fold
- Badge confiance au-dessus du fold (livraison, garantie, retours)
- CTA unique et fort ("Commander", "Acheter", "Essayer", "Rejoindre")

### Product showcase
- Image produit pre-generee (fournie via le contexte) ou placeholder CSS elegant
- 3 benefices produit specifiques (pas generiques)
- Comment ca marche (3 etapes illustrees)
- Ingredients / composition / materiaux si pertinent

### Trust e-commerce
- Garantie satisfait ou rembourse
- Mentions de qualite specifiques au produit
- Temoignages clients avec prenom et detail concret
- Badges paiement securise

### Social proof
- Chiffres specifiques si disponibles
- Temoignages avec details precis (pas "J'adore ce produit !")
