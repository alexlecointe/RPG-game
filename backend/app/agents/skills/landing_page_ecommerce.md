# Skill: Landing page — E-commerce

Applique les instructions generiques de landing_page, plus ces specificites e-commerce :

## Analyse supplementaire

### Conversion e-commerce
- Hero avec produit star en grand visuel haute qualite
- Prix barre + prix promo bien visible
- Badge livraison gratuite au-dessus du fold
- Urgency timer (stock limite, offre temporaire)

### Product showcase
- Gallery produit multi-angles avec zoom
- Variations couleur/taille selectionnables
- Avis clients inline sous chaque produit
- Video produit ou 360° si pertinent

### Trust e-commerce
- Moyens de paiement affiches (CB, PayPal, Apple Pay)
- Politique retours gratuits mise en avant
- Delais de livraison clairs et visibles
- Badges securite (SSL, paiement securise, garantie)

### Social proof
- Nombre de ventes affiche ("2 000+ vendus")
- Photos UGC (User Generated Content) de vrais clients
- Note moyenne produit avec nombre d'avis
- Temoignages clients avec photo et prenom
