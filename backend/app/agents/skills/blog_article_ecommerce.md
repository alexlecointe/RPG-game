# Skill: Article de blog — E-commerce

Applique les instructions generiques de blog_article, plus ces specificites e-commerce :

## Analyse supplementaire

### SEO e-commerce
- Keywords transactionnels ("acheter X", "meilleur X pas cher")
- Keywords informationnels ("comment choisir X", "guide X")
- Ciblage par categorie produit et longue traine shopping
- Optimisation pour les rich snippets produit (prix, avis, stock)

### Contenu e-commerce
- Guides d'achat complets par categorie
- Comparatifs produits avec tableau recapitulatif
- Tutos d'utilisation des produits vendus
- Listes saisonnieres ("Top 10 cadeaux Noel", "Essentiels ete")

### Conversion dans l'article
- Liens produits inline naturellement integres au texte
- CTA "Acheter" ou "Voir le produit" visuellement distinct
- Images produit avec prix affiches dans l'article
- Widgets produit embeddés dans le contenu

### Maillage interne
- Liens vers les pages categories pertinentes
- Liens vers les fiches produit mentionnees
- Liens vers les articles complementaires du blog
- Structure en silo thematique (categorie > sous-categorie > articles)
