# Skill: Design de marque — App mobile

Applique les instructions generiques de brand_design, plus ces specificites app mobile :

## Analyse supplementaire

### App icon design
- Guidelines icone : simplicite, forme distinctive, pas de texte
- Lisibilite en petite taille (29px a 1024px)
- Coherence avec les conventions OS (iOS rounded rect, Android adaptive)

### UI design system
- Composants UI : boutons, inputs, cards, listes, navigation
- Palette dark/light mode (couleurs adaptees aux deux modes)
- Typographie mobile (tailles, hierarchie, lisibilite ecran)
- Spacing et grille (systeme 8pt, safe areas, marges)

### Store presence
- Screenshots style (cadrage, annotations, couleur de fond)
- Video preview branding (intro/outro, transitions, duree)
- Feature graphics (banniere Google Play, App Store header)

### Micro-branding
- Splash screen (animation logo, duree, transition)
- Animations loading (skeleton, spinner, progress custom)
- Empty states (illustrations, ton, CTA)
- Notifications style (icone, son, vibration, groupement)
