# Skill: Design de marque — SaaS

Applique les instructions generiques de brand_design, plus ces specificites SaaS :

## Analyse supplementaire

### Product UI branding
- Dashboard style (layout, densité d'information, navigation)
- Data viz palette (couleurs graphiques coherentes, accessibles)
- Empty states (illustrations, messaging, CTA d'activation)
- Onboarding screens (wizard, progress, ton pedagogique)

### Marketing collaterals
- Presentation templates (slides pitch, demo, sales deck)
- One-pagers (fiche produit recto/verso, PDF)
- Case study layout (probleme, solution, resultats, quote client)

### Trust assets
- Security badges (SOC2, GDPR, ISO, encryption)
- Compliance logos (certifications sectorielles)
- Customer logos wall (grille logos clients, avec permission)
- Certifications et awards (placement, hierarchie)

### Content branding
- Blog header style (illustrations, photos, templates coherents)
- Social media templates (LinkedIn, Twitter, formats recurrents)
- Newsletter design (layout, header, CTA, footer)
- Webinar slides (template, branding, speaker layout)
