# Skill: Growth loop — App mobile

Applique les instructions generiques de growth_loop, plus ces specificites app mobile :

## Analyse supplementaire

### Viral mechanics
- Invite friends in-app avec deep links personnalises
- Partage d'achievements, scores ou creations sur les reseaux
- Deep links qui ramenent directement dans le contexte de l'app
- Incentive double (parrain et filleul recompenses)

### Engagement loop
- Push notifications contextuelles et personnalisees
- Streaks avec recompenses pour usage quotidien
- Daily rewards ou bonus de connexion
- Challenges limites dans le temps (hebdo, mensuels)

### Community loop
- Leaderboards et classements entre utilisateurs
- Features sociales (profils, follow, feed d'activite)
- User-generated content partageable dans et hors de l'app
- Systeme de badges et de niveaux visibles par les autres

### Platform leverage
- Widget iOS pour engagement passif sur le home screen
- Siri shortcuts pour les actions frequentes
- App Clips (iOS) pour decouverte sans installation
- Android instant apps pour essai sans telechargement
