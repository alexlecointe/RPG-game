# Skill: Landing page — SaaS

Applique les instructions generiques de landing_page, plus ces specificites SaaS :

## Analyse supplementaire

### Demo/Trial CTA
- CTA principal "Start free trial" ou "Book a demo" bien visible
- Sans carte de credit si possible ("No credit card required")
- Double CTA : self-serve trial + demo avec sales

### Feature comparison
- Tableau features par tier (Free, Pro, Enterprise)
- Pricing toggle mensuel/annuel avec economie affichee
- Tier "best value" mis en evidence visuellement

### Trust B2B
- Logos clients reconnus (barre de logos)
- Case studies avec metriques de resultats
- Certifications securite (SOC2, GDPR, ISO 27001)
- Uptime SLA affiche (99.9%)

### Integration ecosystem
- Logos des integrations compatibles (Slack, Zapier, etc.)
- API mentionnee pour les profils techniques
- "Works with your existing stack" comme message cle
